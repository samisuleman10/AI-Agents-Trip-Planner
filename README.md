# AI-Agents-Trip-Planner
This project demonstrates how to build an agentic AI system that can reason, plan, and generate a travel itinerary using multi-step workflows and tool usage.


🚀 Overview

Instead of simple one-shot prompting, this system follows a structured pipeline where the AI:

reasons step by step
uses tools (mocked APIs)
applies constraints
refines results iteratively
⚙️ How It Works
Collect user preferences
Check weather data
Fetch available activities
Filter and select best options
Generate initial itinerary
Evaluate and refine (feedback loop)
Generate final itinerary

🧠 Key Concepts
Chain-of-Thought (CoT) reasoning
ReAct (Reason + Act)
Prompt chaining
Tool usage (APIs)
Feedback loops
